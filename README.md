思源笔记【**自用**】主题：

1.   美化界面 UI，追求【**冷淡简约**】的风格，【**聚焦**】文字

2.   隐藏个人不使用的功能：

     1.   书签
     2.   反链
     3.   标签
     4.   文档内部 h1、h6 标签被隐藏， h2 ~ h5 可正常使用。
     5.   ... 等等

3.   切换其他功能面板，请使用快捷键。原 UI 上的图标都被隐藏了

4.   使用本主题，建议先将主题面板排列 - 布局等恢复到【**默认**】状态
